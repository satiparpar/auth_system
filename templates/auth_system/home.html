Hi, This is authentication system which uses Django.
you can use it for your websites.
Each user can register, login, logout.
there are 6 kinds of users:
1- superusers ( full access to website ) you can see this page if you are superuser
2- staff users you can see this page if you are staff user
3- golden users you can see this page if you are golden user
4- silvern users you can see this page if you are silvern user
5- normal users you can see your info in this page if you have registered before
6- Not registered users ( Minimum access to site facilities )

{% if user.is_authenticated %} Hi {{ user.username }}

You can Log out if you like. {% else %} Login {% endif %}
